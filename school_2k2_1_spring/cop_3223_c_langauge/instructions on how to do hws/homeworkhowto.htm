Opening commentary for your C program:

As discussed in class, we are expecting a block of comments at the beginning of your program. In addition to being a useful habit to form, this will be helpful to the graders if there are problems in connecting files received by email with their authors.

The first few lines of your program should resemble the sample program: http://www.cs.ucf.edu/courses/cop3223-1-spr02/CommentedHello.htm

Now you are ready to email a copy of the program to the grader.

 

To create, compile, and run a C program:

1. Type the program using any text editor.

2. Save the program (with name such as: filename.c)
To make it easy for the grader, name the file with your full account number followed by an underscore '_' and the assignment number.  (Example: ama55555_1.c)

3. Compile the program (with command such as: gcc filename.c)

4. Fix errors (using a text editor) and save changes, then compile again.
(Repeat this step as many times as needed)

5. Run the program by typing the execution file name (typically a.out)

6. If the output is incorrect, repeat Steps 4 and 5 as many times as needed.
 

To rename a C program:

If you have your program created under a name other than the filename suggested above, you will want to rename it before submission. For example, let's suppose you have a file in your directory called hw1.c and you want to submit it. If your id is joe01225, you will want to call the new file joe01225_1.c

To rename the file to joe01225_1.c from hw1.c. At the command prompt, type:
mv joe01225_1.c hw1.c

Now you are ready to email a copy of the program to the grader.

To make a printable copy of output:

1. At the command prompt, type:
script output.txt

2. Run the program (typically using a.out)

3. At the command prompt, type: exit

4. This creates a file called output.txt. (You can use another name).
 

To print a file called filename.ext:
At the command prompt, type:
lpr -Pprinter1 filename.ext
 

To Email a copy of the program to the grader:

1. At the command prompt, type: pine
(This will start the Pine Email application)

2. Press the key 'c' to compose a message.

3. Where it says "TO:" type: cop32231

4. Using arrow keys, move down to the field "Attachmnt:"

5. Press Ctrl-t

6. Use arrow keys to highlight the file you are sending.

7. Press enter.

8. Using arrow keys, move down to the body of the message.

9. Type your full name and assignment number.

10. Press Ctrl-x

11. Press 'y'

12. To quit, press 'q'
