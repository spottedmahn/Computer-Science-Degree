5. Determine which of the following are reducible over GF(2)

            a. x3 + x2 + 1

            b. x7 + 1

Solution:

  1. Not reducible, because (x3 + x2 + 1) = x2(x + 1) + 1
  2. Reducible, because (x7 + 1) = (x + 1)(x6 + x5 + x4 + x3 + x2 + x + 1).

Solution:

The guests submit their orders to the host.

The host uses the phone to call the pizza shop.

The order clerk picks up the phone, and talks with the host.

The order clerk tells the pizza cook what to make.

 

The pizza cook tells the order clerk that the order is ready.

The order clerk asks the delivery van to deliver the pizza.

The delivery van delivers the pizza to the host.

The host distributes the pizza to the guests.
