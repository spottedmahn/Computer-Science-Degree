In the first assignment you have implemented DES password-based encryption of an input stream in Java. In this assignment, you will use java.net package (do not confuse with Microsoft’s .NET framework J ) to create a client-server model with a secure communication channel.

 

The server program should wait for incoming TCP connection from the client on port 5000 (use other port for testing if not available), then establish a connection, receive and display client’s incoming data in its window, and send back (echo) all the data that comes from the client. All network communications should be DES-encrypted using the common password that is hard-coded in the server, but is entered in the client program as it starts.

 

The client program should take server IP address as an argument, ask the user for a password, then establish TCP connection with the server at specified IP address and port 5000. The user then can type in any text. Every time [Enter] key is pressed, the string is encrypted and sent to the server. The server decrypts the string, displays it in its own window, then encrypts it and echoes it back to the client. The client receives reply from the server, decrypts it, then displays it on the screen, and awaits the user’s next string of input. When string is “quit”, the server should shut down automatically, then the client shuts down. 

 

To run it on your home computer, make sure that you are connected to the Internet (if using a modem) or have an Ethernet interface available. Also, disable any firewall software for the duration of testing the program. Use IP address 127.0.0.1 to specify localhost (your computer). Open 2 separate DOS prompts (or UNIX terminal windows) to run server and client respectively.

 

Reuse the DesEncrypter class from Assignment 1 if suitable. Hint: modify DesEncrypter’s encode and decode methods to work on Strings: public String encode (String s);
